# BadassPrinces

BadassPrinces started as an idea for a small game jam project built around puzzle solving mechaincs and over-the-top characters, enviroment and animation.

The goal was to experiment with simple combat mechanics, character abilities, and responsive controls while trying out different gameplay ideas in a short development window.

The project was never fully finished, but it served as a fun prototype and a small sandbox for testing combat systems and gameplay mechanics.
